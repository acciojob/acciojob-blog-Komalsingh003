Time, Space, and Complexity

Time complexity is a measure of the amount of time taken by an algorithm to run as a function of the length of the input. An efficient algorithm will have a lower time complexity.

Space Complexity is the amount of memory space an algorithm needs to take as a function of the length of the input. An example is an algorithm with a space complexity of Space Complexity O(n).
